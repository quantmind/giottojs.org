If you use NPM, npm install d3-quant. Otherwise, download the latest release. The released bundle supports AMD, CommonJS, and vanilla environments. Create a custom build using Rollup or your preferred bundler. You can also load directly from https://assets.quantmind.com:

<script src="https://assets.quantmind.com/d3-quant/0.1.0/d3-quant.js"></script>
<script src="https://assets.quantmind.com/d3-quant/0.1.0/d3-quant.min.js"></script>

In a vanilla environment, a d3_quant global is exported. Try d3-quant in your browser.
